# KEY1_He-Lemma-etal
Code used to analyze and process data for manuscript

Information for nature research code and software submission for
Kinase KEY1 controls pyrenoid condensate size throughout the cell cycle by disrupting phase separation interactions

Shan He*, Linnea M. Lemma*, Alejandro Martinez-Calvo, Guanhua He, Jessica H. Hennacy, Lianyong Wang, Sabrina L. Ergun, Ashwani K. Rai, Colton Wang, Luke Bunday, Angelo Kayser-Browne, Quan Wang, Clifford P. Brangwynne, Ned S. Wingreen, Martin C. Jonikas

All code written and run in MATLAB 2022a
Tested both Mac OS and PC Windows
Run in Windows
Install MATLAB according to MathWorks instructions for your OS
Instructions in .info files in each subfolder
Expected output are the figures in the manuscript indicated by the folder name
Run time for each script is ~ 1 minute depending on computer hardware
Test data can be found in each folder and in the excel file with the plot data
